Descripción del Proyecto

Creación de proyecto nuevo con Angular

ng new nombre-del-proyecto (en este caso TP1)

Creación de componente nuevo:

ng g c informacion (o nombre del componente)

Descripción de la función de cada carpeta:

/app: contiene todos los componentes, servicios y módulos principales de la aplicación.

Component: cada componente tiene su .ts, .html, .css y .spec. (este último eliminado).

/screens: contiene las pantallas principales de la aplicación.

/app.routes.ts -> Rutas asociadas con su path y su component

/app.html -> la estructura visual

/app.css -> estilos exclusivos para esta pantalla, no para los componentes

/app.config.ts -> configuración específica relacionada con las rutas

/app.ts -> lógica principal con selector app-root e imports de links de navegación

/public: en esta carpeta funcionan archivos estáticos que deben quedar tal cual

/assets: en esta carpeta van archivos estáticos, fuentes, JSON, íconos
